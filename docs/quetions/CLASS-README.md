## Что такое класс?

Класс является фундаментальной единицей ООП. Класс представляет собой какого-то представление или шаблон 
для объектов. Класс определяет поля и методы. Класс говорит, какого типа объекты мы создаем.

Например, класс `Department`, говорит что мы можем создавать объекты типа `Department`. 

Мы можем создать любое количество объектов класса `Department`.

Все программные конструкции находятся внутри класса

Когда `JVM` начинает компиляцию, код должен содержать как минимум один класс с методом `main`

Класс начинается с обозначения `class`

Файлы с классом должны быть название как класс и с форматом `.java`

```java
class Person{
    private int age;
}
```
